## Функции и возможности на момент версии 1.1.0
1. Инициализация сервера
	- Подгружает конфигурации из JSON'ов.
	- Синхронизирует участников сервера с базой данных при старте/перезапуске.
	- Синхронизирует текстовые/голосовые каналы сервера с базой данных при старте/перезапуске.
	- Обновляет/отправляет динамические сообщения (хабы/центры/узлы) при старте/перезапуске.
	- Обновляет/создает слеш-команды внутри Discord при старте/перезапуске.
	- Формирует сообщения о правилах и прочие статические сообщения.

2. Взаимодействие с участниками
	- Регистрирует новых пользователей в базе данных при первом заходе на сервер.
	- Удаляет пользователей из базы данных при выходе с сервера (статистика не сохраняется при выходе с сервера намеренно).
	- Формирует и отправляет приветственное сообщение новым участникам сервера.
	- Выдает роль "неавтаризированный" новым участникам сервера.
	- Снимает роль "неавторизированный" при успешной авторизации участника.
	- Выдает стартовые роли при успешной авторизации участника.
	- Меняет цвет имени на сервере путем изменения роли участника по выбору самого участника.
	- Сохраняет статистику активности участников (время в голосовых каналах, количество сообщений в текстовых каналах).
	- Сохраняет историю сообщений участников на сервере.
	- Создает динамические личные голосовые комнаты для участников через пораждающие каналы.
	- Следит за обновлениями никнеймов и аватарок участников и уведомляет администрацию.
	- Формирует логи действий участников на сервере (удаление/редактирование сообщений, заход/выход из голосовых каналов и т.д.).

3. Взаимодействие со сторонними сервисами
	- Взаимодействие с OpenAI API для предоставления возможностей ИИ и формирование аналитической статистики на основании истории сообщений пользователей(чат-бот, генерация изображений и т.д.).
	
4. Метрики
	- Подсчет количества сообщений.
	- Подсчет количества отправленных `.gif` изображений.
	- Подсчет количества отправленных `.png` изображений.
	- Подсчет количества времени, проведенного в голосовых каналах.
	- Подсчет количества добавленных реакций на сообщения.
	- Подсчет количества отправленных стикеров.
	- Подсчет количества отправленных `slash`-команд.
