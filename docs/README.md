# Amnyam - Malenkie Discord App

 **Amnyam** - это полнофункциональное backend‑решение на платформе `.NET 10`, интегрированное с `Discord API`. Приложение автоматизирует администрирование сервера __Malenkie__, обеспечивает глубокую аналитику пользовательской активности и расширяет возможности сообщества за счёт ИИ‑модуля (**OpenAI API**).
 
## Информация о релизе

- **Текущая версия**: `1.2.0 - b`
- **Последнее обновление**: `19.01.2026`
- **Статус**: `Активная разработка` 

## Технологический стек
- **Платформа**: `.NET 10.0`.
- **СУБД/ORM**: `EntityFrameworkCore.PostgreSQL 10.0`.
- **Discord API**: `Discord.Net 3.18.0`.
- **OpenAI API**: `OpenAI 2.8.0`.
- **Паттерн MediatR**: `MediatR 14.0.0`.
- **Логгирование**: `Serilog 4.3.0`.
- **Тестирование**: `xUnit 2.9.3` 
- **Деплой**: `Linux Docker`.
